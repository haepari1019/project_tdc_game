## 정체성 A 「초월 (Overdrive)」 — 게이지 → 강화 버스트 윈도우
**운영 의도:** 스킬·평타 명중마다 초월 게이지↑, 가득 차면 일정 시간 **모든 스킬이 강화 발동**. 게이지는
평소(힐·탱 운영) 중에도 쌓이고, **초월 발동 순간 딜러로 스왑**해 몰아친 뒤 빠지는 루프.

- **게이지:** 0~100. 명중당 충전(조작/AI 공통·ungated): 평타 +8, 서브 **명중 적당** +12(광역이 빨리 참).
- **발동:** 가득 → **초월 6.0s**. 창 동안 게이지 0으로 감소, 종료 후 재충전으로만 재발동.

### 강화 = **효과 변화**(단순 배수 아님, 레퍼런스 = LoL 카르마 Mantra)
초월 중 각 서브가 **질적으로 다른 스킬**로 발동. 서로 다른 차원(장판/CC/하드CC)을 더해 **콤보**가 됨.

| 서브 | 기본(AoE) | **초월 변형** |
|---|---|---|
| Q 작열 폭발 | 순간 원형 화염 | **「겁화」** — 폭발 + 명중한 **적에게 화상(Ignited) DoT** 부여(수초 지속딜). **장판 아님 = 대상 한정이라 아군 무피해**(zone은 아군도 피해라 예외처리 시 혼란 → 상태 DoT로 회피). 순간광역 → 지속딜 |
| E 절단 광선 | 직선 관통 빔 | **「중력 광선」** — 빔에 맞은 적을 **빔 중심선으로 끌어당김**(pull) + 빔 폭 확대. 라인딜 → 군집 CC |
| R 빙결 파동 | 원형 냉기 + 감속 | **「절대영도」** — 감속(Chilled)이 **빙결(속박·Rooted)**로 격상 + 반경 확대. 소프트CC → 하드CC |

**콤보(초월 6초 로테이션):** E 중력광선으로 **끌어모으고** → R 절대영도로 **얼리고** → Q 겁화 장판으로 **지진다**.
- 오버레이 delta `overdrive_charge`(Q/E/R): 명중 시 게이지 충전 + (초월 중이면) 해당 서브의 초월 변형 실행.
- **가시화:** HP바 아래 얇은 금색 게이지 바 + 초월 중 금색 발광/뱃지 「초월」.

## 정체성 B 「혈풍 (Blood Gale)」 — 생명 대가 광역 흡수
**운영 의도:** 모든 **서브**가 고정 HP 소모, 단 **명중 적 수 비례 회복**(3기+면 순이득). 서브가 이미 광역이라
많이 맞출수록 유지, 적 소수면 손해 → **누커로 갈아타게** 유도하는 파티 운영 레버.

- **HP 소모:** 서브 시전당 **max_hp 12%**(고정). 평타 무료. 자살 불가 — **최저 1 HP 클램프**.
- **회복:** **명중 적 1기당 max_hp 5%**. 순증감: 1기 −7% / 2기 −2% / **3기 +3% / 4기 +8%**.
- **집계:** delta `blood_soak`(Q/E/R): 시전 시 소모, 발현 후 **서브가 실제로 때린 적 수**(radius/cone)로 회복 정산. 서브가 애초에 광역이라 강제 스플래시 불필요.
- **서브별 변형(kind 분기 — 초월과 대칭, 균일 세금 아님):**
  | 서브 | 혈풍 변형 |
  |---|---|
  | Q 작열(fire) | **흡수 폭발** — 기본 회복(명중 적 비례). 균형형 |
  | E 절단 광선(beam) | **흡혈 광선** — 채널 사이펀이라 회복 ×`beam_refund_mult`(더 빨아옴). 최대 지속형(단 시전 중 고정=리스크) |
  | R 빙결(cold) | **혈빙** — 회복하되 **max_hp 초과분 = 임시 보호막**(`shield_dur`). 방어형 페이오프 |
- **가시화:** 소모 붉은 −N + 회복 초록 +N(혈빙은 청색 +N(막)). 「혈풍」 뱃지.
- **주(파일럿 근사):** 흡혈 광선의 사이펀은 채널 매 틱 훅 대신 시전 시 회복 증폭(lump)으로 근사 — 결속 델타가 cast 후 1회 적용이라. 매 틱 시각 회복은 beam_channel 개조 시 후속.
