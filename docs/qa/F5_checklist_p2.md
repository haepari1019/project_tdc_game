# F5 Manual QA Checklist — Phase 2 (헤드리스 불가 거동)

> **목적:** ci_smoke(컴파일·부팅·로직)로 못 잡는 **실거동**을 F5로 확인하는 수동 체크리스트. S7 QA "헤드리스→F5 브리지" 산물.
> **범위:** 2026-06-26 세션(허브 효과 실연동 · S6b 잔여 · 이연 조각). 각 항목 = 게임 실행 후 확인할 동작.
> **갱신:** 새 거동 기능 추가 시 항목 추가. 자동화 가능해지면 해당 항목을 스모크로 이관 + 여기서 제거.

## A. 허브 효과 실연동
- [ ] **chapel passive** — 성소 T1 승급 후 출격 → 파티 스탯 증가(Tank +15% HP·Healer +12% HP·DPS/Nuker +12% 평타). 승급 전 대비 비교.
- [ ] **armory 기어 상점** — 무기고 T1 승급(Q-HUB-020 선행) → "필기소·상점" 패널 **무기고 섹션**에 B세트 표시 → 구매(ward_scrap) → 스태시에 들어옴. T2 → C세트 추가.
- [ ] **소모품 상점** — 패널 "보급" 섹션 → 부활 스크롤 20 scrap 구매 → 스태시 소모품 +1.
- [ ] **per-AB tier / scribe_shop T2** — 상점에서 Advanced 생본은 scribe_shop T1선 "T2 필요"로 잠김 → scribe_shop T2 승급 후 구매 가능. 스킬북 툴팁에 "등급" 색구분(Basic 회색·Advanced 청색·Master 금색).

## B. capacity (성장 압력 축)
- [ ] **군수 런 운반 한도** — 런 중 carry를 한도(T0=12)까지 채운 뒤 추가 루트 픽업 시도 → 거부(드롭이 바닥에 남음, 콘솔 `[CARRY]`). 군수 승급(14/16) → 더 담김.
- [ ] **창고 영속 한도** — 허브에서 스태시를 한도(T0=20)까지 채운 뒤 백팩→스태시 입금 시도 → 거부(`창고 한도 초과` 메시지, 아이템 원위치). 상점 구매 버튼도 비활성. 창고 승급(28/36) → 더 담김.
- [ ] **비파괴 확인** — 기존 세이브 로드 시 over-cap이어도 아이템 유실 0(추가만 차단). 신규 게임 시드 = 19개(여유 1).

## C. 평타 특수거동 (ba 아키타입)
- [ ] **cleave** — `gear_ward_tank_march_plate`(march_stomp) 장착 → 평타가 대상 주변 적까지 splash(2차 60% 피해).
- [ ] **knockback** — 같은 march/aegis_ram 평타 → 피격 적이 밀려남.
- [ ] **pierce** — `gear_ward_dps_weave_staff`(weave_lance) 장착 → 평타가 일직선상 적들을 관통(2차 70%).
- [ ] **단일타 회귀** — 스타터 anchor_bulwark(shield_tap) → 단일 대상만(거동 없음).

## D. 적 kit (AB-003/005/007 적측)

- [ ] **AB-005 연타** — EN-010(근접 fodder) 근접 연타 시그니처.
- [ ] **AB-007 후퇴** — EN-005(nuker)가 **HP≤50% + 근접 위협** 시 적 반대로 후퇴 도약(disengage).
